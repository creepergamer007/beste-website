Aan deze pagina mag iedereen zijn of haar content toevoegen.

De spelregels:

  • Nieuwe content wordt altijd toegevoegd via goedgekeurde pull requests.
    • Lees de theorie in het lesmateriaal om te lezen hoe dat werkt.
  • Nieuwe stukken HTML code voeg je toe in een eigen section-element.
    • Op dezelfde wijze als de al bestaande section(s).
  • De code hoeft niet volgens de alom geldende goede smaak of "best practices" te zijn.
    • Zolang je code maar functioneert en de rest van de pagina niet breekt.
  • Je mag eerdere stukken code van andere developers niet verwijderen.
    • Je eigen content mag je natuurlijk wel aanpassen achteraf, via een nieuwe pull request.
  • Je toegevoegde content hoeft niet per se (alleen) uit tekst te bestaan.
    • Afbeeldingen, animaties, hyperlinks en video's zijn ook prima.
  • Discriminerende, beledigende of NSFW content is niet toegestaan.
    • Pull requests die dergelijke inhoud bevatten worden per definitie geweigerd.
  • Als je weet wat je doet mag je ook CSS gebruiken toevoegen.
    • Zorg er in dat geval wel voor dat je alleen je eigen section aanspreekt middels een id.
